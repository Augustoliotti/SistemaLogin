# SistemaLogin

Projeto de exemplo em ASP.NET Core MVC para autenticação com cadastro e login de usuários.

Principais comandos

Build:
```
dotnet build
```

Executar:
```
dotnet run
```

Recomendações antes de subir ao GitHub:
- Remover ou não commitar credenciais em `appsettings.Development.json`.
- Adicionar `LICENSE` se for open-source.
